### MS/MSスペクトル類似度、ネットワーキング

- 解析内容
  - MassBankのmspファイルに含まれるMS/MSスペクトルに対して、同じ物質のものは統合し、MS/MSスペクトルの類似度を計算する
  - 得られたスペクトルの類似度からネットワークを描く
- 解析例
  - https://hiroyukiyamamoto.github.io/doc/msinfo/Network/network.html
- 検討事項
  - 内積以外のMS/MSスペクトル類似度について
